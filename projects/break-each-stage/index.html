Break Each Stage

Python pygame으로 제작한 2D 아케이드 게임 프로젝트

Source code

1. 프로젝트 개요

  • Python pygame으로 제작한 2D 아케이드 게임 프로젝트
  • 고전 게임 세상에서 가장 어려운 게임에서 아이디어를 얻어, 장애물을 피해 스테이지를 클리어하는 방식으로 구현
  • 사용자가 직접 맵을 제작할 수 있는 툴까지 함께 개발해 스테이지를 확장할 수 있도록 구성

주요 기능

  • 방향키 기반 플레이어 이동
  • 고정 장애물 / 이동 장애물 충돌 판정
  • 게임 오버 / 스테이지 클리어 / 다음 단계 이동
  • 스테이지 데이터 저장 및 불러오기
  • 사용자 제작용 맵 에디터 제공

2. 담당 역할

개인 프로젝트(1인 개발)

  • 게임 로직 전체 구현
  • 스테이지 저장 / 로딩 구조 설계
  • 이동 장애물 동작 및 충돌 판정 구현
  • 맵 제작 툴 별도 구현

3. 사용 기술

Python

  • 처음 배우던 언어로 직접 게임을 만들며 구조화 경험을 쌓기 위해 사용

Pygame

  • 2D 화면 구성, 입력 처리, 충돌 판정 구현에 활용

Text File I/O

  • 맵 데이터를 저장하고 다시 불러오는 방식에 사용

4. 구현 사항

  • 플레이어 블록 이동 및 도착 판정 구현
  • 장애물 충돌 시 게임 오버 처리
  • 스테이지 완료 후 다음 단계로 넘어가는 흐름 구현
  • 고정 블록 / 이동 블록 정보를 파일로 저장하는 구조 구현
  • 키보드와 마우스로 맵을 직접 제작할 수 있는 에디터 구현

5. 문제 해결 사례

상황

스테이지를 늘릴수록 맵을 코드로 직접 수정하는 방식이 비효율적이었다.

해결

맵을 별도 파일로 저장하고, 사용자가 직접 블록을 배치할 수 있는 맵 제작 툴 추가 개발

성과

게임과 콘텐츠 제작을 분리해 새로운 스테이지를 쉽게 추가할 수 있는 구조 확보

의미

플레이 가능한 게임과 확장용 툴을 함께 만들며 사용성과 확장성까지 고민한 경험
