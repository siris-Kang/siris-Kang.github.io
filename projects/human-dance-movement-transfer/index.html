Human Dance Movement Transfer via First Order Motion Model

First Order Motion Model 기반 전신 춤 동작 전이 연구

1. 프로젝트 개요

  • First Order Motion Model 기반 전신 동작 전이 연구
  • 한 사람의 정지 이미지에 다른 사람의 동작 영상을 전달해 새로운 동작 영상을 생성하는 연구
  • 복잡한 자세 라벨링 없이 keypoint 기반으로 움직임을 학습하는 모델 활용
  • 단순 논문 재현이 아니라, 전신 동작에 더 맞도록 입력 표현, keypoint 수, 학습 환경을 조정하는 것을 목표로 진행

주요 기능

  • source image + driving video 기반 동작 전이
  • keypoint 추출 및 motion representation 생성
  • source appearance를 유지한 채 동작 반영 이미지 생성
  • 프레임 단위 결과를 영상으로 생성
  • 전처리 및 학습 환경 조정을 통한 전신 동작 표현 개선

2. 담당 역할

  • 논문 구조 및 코드 분석
  • 데이터셋 전처리
  • bounding box 기준 조정
  • keypoint 수 10개 → 14개 수정
  • mixed precision training 적용
  • 학습 수행 및 결과 확인

3. 사용 기술

PyTorch

  • 모델 학습 및 코드 수정에 사용
  • 기존 연구 구현을 분석하고 실험 조건을 바꾸기 적합했음

First Order Motion Model

  • 별도 자세 라벨 없이 motion transfer가 가능한 구조여서 사용
  • 전신 동작 전이 문제를 다루기에 적합했음

Mixed Precision Training

  • GPU 메모리 부담을 줄이고 학습 효율을 높이기 위해 적용
  • 제한된 자원에서도 학습이 가능하도록 개선

Python

  • 데이터 전처리, 학습 코드 수정, 실험 관리에 사용

4. 구현 사항

  • First Order Motion Model의 keypoint detector / dense motion / generator 구조 분석
  • 전신 동작 표현에 맞게 keypoint 수를 10개에서 14개로 확장
  • 전신이 더 일관되게 들어오도록 bounding box 기준 조정
  • 학습 자원 부담을 줄이기 위해 mixed precision training 적용
  • 수정한 조건으로 학습을 수행하고 결과 비교 확인

5. 문제 해결 사례

1) GPU 자원 부담으로 학습이 비효율적이던 문제

상황

생성 모델 특성상 학습 시 메모리 사용량이 커 실험 부담이 컸다.

해결

  • mixed precision training 적용
  • 일부 연산을 낮은 정밀도로 처리해 GPU 자원 사용량 절감

성과

  • 제한된 자원 환경에서도 학습 수행 가능
  • 학습 효율 개선

의미

  • 단순 모델 사용이 아니라 학습 환경 자체를 최적화한 경험 확보

2) 전신 동작 특징이 충분히 반영되지 않는 문제

상황

기존 keypoint 수와 입력 전처리만으로는 전신 움직임 표현이 부족하다고 판단했다.

해결

  • keypoint 수를 10개 → 14개로 확장
  • bounding box 기준 조정으로 전신 입력 일관성 개선

성과

  • 전신 동작을 더 잘 반영할 수 있는 입력 구조로 개선
  • 전처리와 모델 설정을 함께 조정한 실험 경험 확보

의미

  • AI 모델 성능은 구조뿐 아니라 입력 데이터 표현에도 크게 좌우된다는 점을 확인

6. 성과 및 결과

  • 전신 동작 전이 모델 구조와 학습 흐름을 직접 분석하고 수정한 경험 확보
  • keypoint 수 조정과 전처리 개선으로 전신 동작 표현에 맞는 실험 수행
  • mixed precision 적용으로 제한된 GPU 환경에서도 학습 가능하도록 개선
  • 단순 재현이 아니라 데이터 표현, 학습 조건, 자원 효율을 함께 조정한 연구 경험 축적
