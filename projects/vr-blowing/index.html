VR Blowing: A Pyhsucally Plausible Air Interaction Method

입으로 바람을 불어 바람개비를 조작하는 VR 인터랙션 연구

0. 프로젝트 소개 영상

VR Blowing 프로젝트 소개 영상

1. 프로젝트 개요

  • 사용자가 입으로 바람을 불었을 때 바람개비와 탁자 위 바람개비가 실제처럼 반응하도록 구현한 VR 인터랙션 연구
  • Unity, C# 기반으로 가상환경에서 바람개비 반응 및 회전 동작 구현
  • IEEE Transactions on Visualization and Computer Graphics 저널 게재, 2저자

주요 개발 내용

  • 바람에 반응하는 바람개비 회전 및 반응 구현
  • 중력, 부력, 항력, 바람의 힘을 반영한 물리 모델 적용
  • VR 사용자 실험 환경 구성 및 평가 진행

2. 나의 역할

  • 관련 논문을 읽고 물리 요소를 분석해 Unity 환경에 구현
  • 중력, 부력, 항력, 외력을 고려한 바람개비 반응 로직 개발
  • 실험 환경 구성 및 사용자 평가 전반 참여
  • IPQ, CAQ, SEQ 설문 및 인터뷰 결과 정리

3. 트러블슈팅

1) 바람개비 반응의 비현실성

문제

초기에는 바람개비 반응이 어색하고 실제 바람에 반응하는 느낌이 부족했음

원인 분석

  • 단순 이동/회전만으로는 바람개비의 자연스러운 반응을 표현하기 어려웠음
  • 실제 반응에는 중력, 부력, 항력, 외력이 함께 작용함을 확인

해결

  • 유체 속 물체의 움직임 관련 논문을 찾아 핵심 물리 요소를 반영
  • Unity에서 중력, 부력, 항력, 바람의 힘을 함께 적용해 더 자연스러운 움직임 구현

2) 구현 결과의 검증 필요

문제

구현만으로는 사용자가 실제처럼 느끼는지 확인하기 어려웠음

원인 분석

물리적으로 맞아 보여도 사용자 체감은 다를 수 있어 정량적 검증이 필요했음

해결

  • IPQ, CAQ, SEQ 설문과 주관식 인터뷰를 포함한 사용자 평가 진행
  • 구현 방식에 대해 바람을 부는 동작이 더 현실적으로 느껴진다는 응답을 85% 확보
