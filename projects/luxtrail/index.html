LuxTrail

스터디원의 백준 풀이 현황을 한 페이지에서 확인하는 웹 프로젝트

1. 프로젝트 개요

  • 스터디원의 백준 풀이 현황을 한 페이지에서 확인하는 웹 프로젝트
  • Solved.ac 비공식 API를 활용해 당일 풀이 여부, 특정 문제 풀이 여부, 당일 랭킹 기능 구현

주요 개발 내용

  • Solved.ac 비공식 API 연동
  • 당일 풀이 여부 확인 기능 구현
  • 특정 문제 풀이 여부 조회 기능 구현
  • 당일 랭킹 기능 구현

2. 나의 역할

  • 서비스 기획
  • API 문서 분석 및 데이터 수집 구조 설계
  • 풀이 여부 확인 / 특정 문제 조회 / 랭킹 기능 구현

3. 트러블슈팅

  • 초기에는 백준 크롤링으로 구현하려 했으나, 정책상 적절하지 않음을 확인하고 해당 방식 제거
  • Solved.ac 비공식 API 기반으로 구조를 전환하여 기능을 재설계
