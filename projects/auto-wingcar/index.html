Auto-wing Car

비행기를 끌어주는 토잉카의 자율주행화 프로젝트

0. 프로젝트 소개

Auto-wing Car 소개 영상

Auto-wing Car GitHub

1. 프로젝트 개요

  • 공항 내 자율주행 토잉카 개발 프로젝트
  • Jetson Orin Nano에서 주행 명령 생성
  • STM32 NUCLEO-F103RB에서 UART로 명령 수신 후 모터/서보 제어
  • 경로계획 + 임베디드 제어 + 안전 정지 구조까지 전체 시스템 구현

주요 개발 내용

경로 생성 알고리즘

  • mask 기반 장애물 맵 생성
  • Hybrid A* 기반 트레일러 경로 탐색 개선
  • Reeds-Shepp 기반 analytic expansion 적용

STM32 펌웨어

  • DC 모터 PWM 제어
  • 조향 서보 / 도킹 서보 제어
  • UART 패킷 통신 구현
  • FreeRTOS 확장

2. 나의 역할

경로계획

  • mask.png에서 장애물 좌표 추출 후 월드 좌표로 변환
  • 비행기 크기를 반영한 inflation 적용
  • Grid A* heuristic map 안정성 개선
  • Hybrid A* 탐색 비용 직접 설계
  • 전진 / 후진과 조향 변화에 따른 패널티 부여
  • jackknife penalty 부여

STM32 펌웨어

  • UART 패킷 기반 속도/조향/도킹 명령 수신
  • PWM 기반 DC 모터 제어
  • 조향 서보 및 도킹 서보 제어
  • Timeout / Enable / E-stop 조건에서 즉시 정지하는 안전 로직 구현

FreeRTOS 구조화

  • SafetyTask / ControlTask / DockTask / UartRxTask 분리
  • 최신 명령만 반영하는 Queue 구조 설계
  • 상위 보드 이상 시 STM32가 최종 안전 정지 수행하도록 구현

3. 트러블슈팅

1) 서보 구동 시 전원 불안정 문제

문제

서보 동작 순간 전원이 불안정해지며 시스템 동작 이상 발생

원인 분석

전원 변동과 서보 부하 영향 확인

해결

  • 토잉카의 전원 공급 구조 재설계 필요
  • 11V 전원 환경으로 변경 후 배선 수정

2) Jetson 역전원 문제

문제

모터 구동 중 Jetson Orin Nano가 꺼지는 문제 발생

원인 분석

모터드라이버에 연결된 전원을 11V로 바꾸면서 젯슨 쪽으로 역전원이 들어감을 확인

해결

스키마틱과 배선 구조를 다시 확인하며 Jetson과 모터드라이버 햇의 전원 라인 분리
