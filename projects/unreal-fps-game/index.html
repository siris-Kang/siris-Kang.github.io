DUSK

언리얼 FPS 게임 개발 프로젝트

0. 프로젝트 소개 영상

DUSK 소개 영상

1. 프로젝트 개요

  • Unreal Engine을 활용해 1인칭 시점에서 플레이할 수 있는 게임 프로토타입을 만든 프로젝트입니다.
  • 단순 장면 제작이 아니라, 직접 이동하고 탐색할 수 있는 레벨을 구성하며 FPS 게임의 기본 흐름을 구현하는 것이 목표였습니다.
  • 환경 구성, 시점 체감, 플레이 동선이 함께 어우러지는 게임 공간을 만드는 데 집중했습니다.

주요 기능

  • 1인칭 시점 기반 플레이 환경 구성
  • 탐색 가능한 3D 레벨 제작
  • 환경 오브젝트와 지형을 활용한 공간 연출
  • 실제 플레이 기준으로 조정한 게임 프로토타입 구성

2. 담당 역할

  • Unreal Engine 기반 맵 구성과 레벨 배치
  • 1인칭 시점에서 자연스럽게 보이는 공간감 조정
  • 환경 오브젝트 배치와 플레이 동선 정리
  • 플레이 테스트를 통한 씬 구성 보완

3. 사용 기술

Unreal Engine

  • 실시간 3D 환경 안에서 직접 플레이 가능한 레벨을 구성하기 위해 사용했습니다.
  • 월드 구성, 조명, 카메라 시점, 플레이 테스트를 한 환경에서 반복하기에 적합했습니다.

Level Design

  • 플레이어가 실제로 이동할 공간과 시각적 포인트를 함께 설계하기 위해 집중한 요소입니다.
  • 정적인 장면이 아니라 플레이 흐름을 고려한 공간 구성이 필요했습니다.

3D Environment Composition

  • 지형과 오브젝트를 조합해 게임 세계의 분위기와 이동 경로를 구성했습니다.
  • 시각 완성도와 플레이 편의성을 함께 맞추는 데 중요했습니다.

4. 구현 사항

  • 1인칭 시점에서 플레이 가능한 공간을 Unreal 환경 안에 구성했습니다.
  • 지형과 환경 오브젝트를 배치해 맵의 분위기와 이동 동선을 설계했습니다.
  • 플레이어 시점에서 어색하지 않도록 오브젝트 크기와 간격을 반복 조정했습니다.
  • 플레이 테스트를 통해 씬이 단순 전시 화면이 아니라 게임 공간으로 느껴지도록 다듬었습니다.

5. 성과 및 결과

  • Unreal Engine을 활용해 직접 이동하고 탐색할 수 있는 1인칭 게임 프로토타입을 구성했습니다.
  • 레벨 디자인과 환경 구성, 플레이 테스트를 반복하며 게임 공간을 다듬는 경험을 쌓았습니다.
  • 정적인 3D 장면 제작을 넘어 플레이 기준으로 공간을 설계하는 감각을 익혔습니다.
