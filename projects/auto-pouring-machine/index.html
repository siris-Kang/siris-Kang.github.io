자동 술 따라주는 기계

아두이노, 무게센서 기반 자동 소주 따르기 임베디드 프로젝트

0. 프로젝트 소개 영상

소주 따라주는 기계 소개 영상

1. 프로젝트 개요

  • 무게센서를 이용해 소주병의 무게를 측정하고, 무게에 따라 병의 기울기를 조절해 술을 따라주는 기계를 제작한 프로젝트입니다.
  • 2인 프로젝트로 하드웨어 디자인, 회로 구성, 프로그래밍 전 과정을 함께 수행하며 실제로 동작하는 임베디드 시스템을 만드는 것이 목표였습니다.
  • 센서 입력과 서보 제어를 연결해 병의 상태에 따라 동작이 달라지는 구조를 구현했습니다.

주요 기능

  • 로드셀 무게센서를 이용한 병 무게 측정
  • 무게에 따라 병 기울기를 조절하는 제어 로직 구현
  • 서보모터를 이용한 병 기울임 동작 구현
  • 하드웨어 및 회로 구성 공동 설계

2. 담당 역할

  • 프로그래밍 전 과정 담당
  • 로드셀 센서 값을 읽고 처리하는 로직 구현
  • 무게에 따라 병의 기울기를 조절하는 알고리즘 설계
  • 서보모터 제어 코드 작성
  • 회로 구성 과정도 함께 고민하며 전체 동작 구조 설계 참여

팀 내 역할 분담(총 2인)

  • 하드웨어 구조 및 회로 구성: 공동 진행
  • 프로그래밍 및 제어 로직 구현: 본인 중심 담당

3. 사용 기술

Arduino

  • 센서 입력과 서보 제어를 빠르게 연결해 실제 프로토타입을 구현하기 위해 사용했습니다.
  • 하드웨어 실험과 코드 수정이 반복되는 프로젝트에 적합했습니다.

로드셀 무게센서

  • 병의 무게 변화를 직접 측정해 동작 기준으로 활용하기 위해 선택했습니다.
  • 기울임 동작을 단순 시간 제어가 아니라 실제 상태 값에 따라 조정할 수 있었습니다.

Servo Motor

  • 병의 기울기를 제어하는 구동부로 사용했습니다.
  • 센서 값과 연결해 각도를 단계적으로 조정하는 구조를 만들기에 적합했습니다.

4. 구현 사항

  • 로드셀 센서 값을 읽어 병 무게를 측정하고, 제어 로직에 반영했습니다.
  • 무게에 따라 병 기울기를 단계적으로 조절하는 알고리즘을 구성했습니다.
  • 서보모터를 이용해 실제로 병을 기울이고 다시 복원하는 동작을 구현했습니다.
  • 회로 연결과 하드웨어 구조를 함께 고려하며 전체 동작 흐름을 맞췄습니다.

5. 문제 해결 사례

1) 센서값에 따라 병 기울기를 어떻게 조절할지 기준이 필요하던 문제

상황

병의 무게가 계속 변하기 때문에, 어떤 기준으로 병의 기울기를 조절해야 실제로 술이 안정적으로 따라지는지 결정하는 과정이 필요했습니다.

해결

  • 병이 가벼워질수록 같은 각도로는 술이 잘 나오지 않을 수 있다는 점에 주목했습니다.
  • 무게에 반비례하도록 병의 각도를 조절하는 방식으로 제어 로직을 설계했습니다.
  • 센서값에 따라 기울기를 단계적으로 조정하도록 구현해 실제 동작 가능성을 높였습니다.

성과

  • 센서 입력과 구동부 제어가 연결된 형태의 프로토타입을 완성했습니다.
  • 단순 수동 조작이 아니라 상태 기반 제어 구조를 직접 구현할 수 있었습니다.

의미

  • 실제 환경에서 동작하는 임베디드 시스템은 센서값을 어떤 기준으로 해석하느냐가 핵심이라는 점을 배웠습니다.
  • 하드웨어 실험과 제어 알고리즘 설계를 함께 맞춰보는 경험을 쌓았습니다.

6. 성과 및 결과

  • 무게센서와 서보모터를 연결한 자동 소주 따르기 프로토타입을 직접 구현했습니다.
  • 하드웨어 구조, 회로 구성, 제어 로직을 함께 고려하는 임베디드 프로젝트 경험을 쌓았습니다.
  • 센서 입력 기반 상태 제어를 실제 동작으로 연결하는 과정을 체감할 수 있었습니다.
